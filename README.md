# odin-recipes

A simple recipe website built as part of [The Odin Project](https://www.theodinproject.com/) Foundations Course.  
This project demonstrates basic HTML skills and introduces the workflow of creating, organizing, and publishing projects on GitHub.

---

## 🚀 Features
- **Homepage (index.html):** Contains links to individual recipe pages.
- **Recipe Pages:** Each page includes:
  - Title and heading
  - Image of the dish
  - Description section
  - Ingredients list (unordered list)
  - Steps section (ordered list)
  - Navigation link back to the homepage
- **Multiple Recipes:** Three recipes are included, linked from the homepage.

---

## 🛠️ Skills Practiced
- Writing semantic HTML
- Structuring content with headings, paragraphs, lists, and links
- Organizing files and directories
- Using Git for version control (clone, commit, push)
- Publishing projects with GitHub Pages
